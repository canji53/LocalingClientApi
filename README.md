# LocalingClientApi

## 概要

* [ローカりんぐ](https://www.loclaing.ml)でクライアントユーザーが使用する API 群
* ローカりんぐは、全国のローカルメディアのコンテンツを収集して一覧化したサイトです

## 技術スタック

* Serverless
* Node.js
* Inoreader API, SendGrid API
* AWS, API Gateway, Lambda, Lambda Layer, DynamoDB
* AWS SAM, AWS CLI
* Git, GitHub, GitHub Actions
